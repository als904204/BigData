사이트 소개

이 웹사이트는 주식 데이터 엑셀 파일을 업로드하여 시각화 및 분석하는데 도움을 주는 웹 사이트 입니다.

1. Colab 런타임 설정

  • Colab 에 접속하여 1. 라이브러리 설치 (finance-DataReader) 부분만 실행합니다
    (약3분가량 소모됩니다)
  • 성공적으로 라이브러리들이 다운로드되면 런타임-다시 시작 버튼 클릭을 클릭합니다
    (마찬가지로 약3분가량 소모되고 해당 동작을해야 정상적으로 라이브러리와 한글 폰트가 적용됩니다)
  • 이제부터 런타임-다시 시작할 필요 없이 원하는 종목을 선택 해 데이터를 다운 받으실 수 있습니다
    (단, Colab 재접속 혹은 런타임이 끊기거나,한글 또는 라이브러리가 적용안될 경우 다시 하는걸 권장합니다)

2. 주식 종목 선택

  • 주식 종목 선택2(>)을 클릭하여 숨겨진 셀을 확장합니다
  • 주식 종목 범위로 찾기2-2에서 범위를 입력합니다 find_stock_by_range(0, 10)
  • 주식 종목 이름으로 선택2-3에서 주식종목을 입력합니다 find_stock_code_by_name('삼성전자')
  • 이제 stock_df2-4 변수에 위에서 선택한 주식 종목 데이터 프레임이 저장됩니다

3. 데이터 전처리

  • null값처리3-1에서 null 값을 처리합니다
  • (시가,고가,저가,거래량) 결측치 전처리3-2에서 결측된 값(0) ['Open', 'High', 'Low', 'Volume']을 처리합니다

4. 데이터 시각화 목록

  • Closing Price
  • Volume
  • High and Low Price
  • Moving Average
  • Candlestick with Moving Averages

5. 예측 값

작성중..

6. 엑셀 파일 다운로드

  • 선택한 종목의 영어 이름으로 엑셀 파일이 저장됩니다.
    (삼성전자->SAMSUNG_data.xlsx, LG에너지솔루션->LG_ENERGY_SOLUTION_data.xlsx)

7. 엑셀 파일 업로드

  • 왼쪽 상단에 원하시는 Chart 메뉴를 선택한 다음 파일선택 버튼을 눌러 다운받은 엑셀 파일을 업로드합니다
  • 취소버튼을 눌러 다른 엑셀 파일을 업로드 할 수 있습니다
